## **PHASE 1 — Computer Fundamentals (Start Here)** 

## **1.1 Introduction to Computer Systems** 

- Evolution of computing devices; components of a computer system and their interconnections 

- Input/output devices, CPU 

- Memory: primary, cache, secondary; units (Bit, Byte, KB, MB, GB, TB, PB) 

- Data deletion, recovery and security concerns 

## **1.2 Types of Software** 

- System software: operating systems, system utilities, device drivers 

- Programming tools & language translators: assembler, compiler, interpreter 

- Application software: generic and specific purpose 

## **1.3 Number Systems & Encoding** 

- Binary, Octal, Decimal, Hexadecimal number systems; conversion between them 

- Encoding schemes: ASCII, ISCII, UNICODE (UTF-8, UTF-32) 

- Number System and Codes (digital perspective) 

## **1.4 Boolean Logic & Digital Circuits** 

- NOT, AND, OR, NAND, NOR, XOR gates; truth tables; De Morgan's laws; logic circuits 

- Combinational: logic gates, simplifications, half/full adders, encoders, decoders, multiplexers 

- Sequential: Flip Flops, RAM and ROM 

## **PHASE 2 — Programming with Python** 

## **2.1 Problem Solving & Algorithms** 

- Steps: analysing, developing algorithm, coding, testing, debugging 

- Flowcharts, pseudocode, decomposition 

## **2.2 Python Basics** 

- Introduction to Python; features; interactive & script execution modes 

- Character set; tokens (keyword, identifier, literal, operator, punctuator) 

- Variables; l-value and r-value; comments 

## **2.3 Data Types & Operators** 

- Data types: int, float, complex, boolean, string, list, tuple, None, dictionary; mutable vs immutable 

- Arithmetic, relational, logical, assignment, augmented assignment operators 

- Identity operators (is, is not); membership operators (in, not in) 

## **2.4 Expressions, I/O & Errors** 

- Operator precedence; expression evaluation; type conversion (explicit & implicit) 

- Accepting input from console; displaying output 

- Error types: syntax, logical, runtime 

## **2.5 Control Flow** 

- Indentation; sequential, conditional and iterative flow 

- Conditional: if, if-else, if-elif-else — programs: absolute value, sort 3 numbers, divisibility 

- Iterative: for loop, range(), while loop; break/continue; nested loops 

- Programs: generating patterns, summation of series, factorial 

## **2.6 Strings** 

- Indexing; concatenation, repetition, membership, slicing; traversal 

- Built-in: len(), capitalize(), title(), lower(), upper(), count(), find(), index() 

- endswith(), startswith(), isalnum(), isalpha(), isdigit(), islower(), isupper(), isspace() 

- lstrip(), rstrip(), strip(), replace(), join(), partition(), split() 

## **2.7 Lists** 

- Indexing; concatenation, repetition, membership, slicing; nested lists; traversal 

- Built-in: len(), list(), append(), extend(), insert(), count(), index(), remove() 

- pop(), reverse(), sort(), sorted(), min(), max(), sum() 

- Programs: max/min/mean; linear search; frequency count 

## **2.8 Tuples** 

- Indexing; operations; nested tuples; tuple assignment 

- Built-in: len(), tuple(), count(), index(), sorted(), min(), max(), sum() 

- Programs: max/min/mean; linear search; frequency count 

## **2.9 Dictionaries** 

- Accessing items by key; mutability; traversal 

- Built-in: len(), dict(), keys(), values(), items(), get(), update(), del, clear() 

- fromkeys(), copy(), pop(), popitem(), setdefault(), max(), min(), sorted() 

- Programs: character frequency; employee salary dictionary 

**2.10 Functions** 

- Types: built-in, module-defined, user-defined 

- Creating functions; arguments/parameters; default & positional parameters 

- Return values; flow of execution; scope — global vs local 

## **2.11 Python Modules** 

- import and from statement 

- math: pi, e, sqrt(), ceil(), floor(), pow(), fabs(), sin(), cos(), tan() 

- random: random(), randint(), randrange() 

- statistics: mean(), median(), mode() 

## **PHASE 3 — File Handling & Data Structures** 

## **3.1 File Types & Paths** 

- Text files, Binary files, CSV files; relative and absolute paths 

## **3.2 Text Files** 

- Open modes: r, rt, w, w+, a, a+; closing files; with clause 

- Writing/appending: write(), writelines() 

- Reading: read(), readline(), readlines(); seek() and tell(); data manipulation 

## **3.3 Binary Files** 

- Open modes: rb, rb+, wb, wb+, ab, ab+ 

- pickle module: dump() and load() 

- Operations: read, write/create, search, append, update 

## **3.4 CSV Files** 

- csv module; open/close CSV files 

- Write: csv.writer() | Read: csv.reader() 

## **3.5 Data Structures & Algorithms** 

- Data structure types: Array, Stack, Queue, Linked List, Heap, Tree, Binary Search Tree 

- Stack: push & pop operations; implementation using list 

- Searching: Linear Search, Binary Search 

- Sorting: Bubble, Selection, Insertion, Quick Sort, Merge Sort, Heap Sort 

- Algorithm Analysis: Big-O (O), Omega (Omega), Theta notations 

- Graphs: connected, regular, bipartite; cycles & circuits; spanning trees; BFS; DFS 

## **PHASE 4 — Operating Systems & Networks** 

## **4.1 Operating Systems** 

- Functions; types: multiprocessing, multitasking, multiprogramming, time sharing, distributed 

- Virtual memory, paging, segmentation, fragmentation 

- Mutual Exclusion, Lock-Unlock, semaphores 

- CPU scheduling, I/O scheduling, resource scheduling, scheduling algorithms 

- Deadlock and its handling 

## **4.2 Computer Networks** 

• Evolution: ARPANET, NSFNET, INTERNET 

- Data communication: sender, receiver, message, media, protocols; bandwidth; data transfer rate 

- IP address; switching: circuit switching, packet switching 

- Wired media: twisted pair, co-axial, fibre-optic; Wireless: radio, micro, infrared waves 

- Devices: Modem, Ethernet card, RJ45, Repeater, Hub, Switch, Router, Gateway, WiFi card 

- Types: PAN, LAN, MAN, WAN; Topologies: Bus, Star, Tree 

- Protocols: HTTP, FTP, PPP, SMTP, TCP/IP, POP3, HTTPS, TELNET, VoIP 

- Web services: WWW, HTML, XML, domain names, URL, web browser, web servers, web hosting 

## **4.3 Websites & Web Browsers** 

- Website vs webpage; static vs dynamic web page; web server and hosting 

- Browsers: commonly used, settings, add-ons, plug-ins, cookies 

## **PHASE 5 — Database Management** 

## **5.1 Database Concepts** 

- Introduction; relational data model: relation, attribute, tuple, domain, degree, cardinality 

- Keys: candidate key, primary key, alternate key, foreign key 

## **5.2 SQL — Structured Query Language** 

- DDL and DML; data types: char(n), varchar(n), int, float, date 

- Constraints: NOT NULL, UNIQUE, PRIMARY KEY 

- Commands: CREATE/USE/SHOW/DROP DATABASE, SHOW/CREATE/DESCRIBE/ALTER/DROP TABLE 

- DML: INSERT, DELETE, SELECT, UPDATE 

- Operators: mathematical, relational, logical; aliasing; DISTINCT; WHERE; IN; BETWEEN 

- ORDER BY; NULL / IS NULL / IS NOT NULL; LIKE; UPDATE; DELETE 

- Aggregate functions: MAX(), MIN(), AVG(), SUM(), COUNT(), COUNT(*) 

- Clauses: GROUP BY, HAVING 

- Joins: cartesian product, equi-join, natural join 

## **5.3 SQL Functions (Extended)** 

- Math: POWER(), ROUND(), MOD() 

- Text: UCASE(), UPPER(), LCASE(), LOWER(), MID(), SUBSTRING(), SUBSTR(), LENGTH(), LEFT(), RIGHT(), INSTR(), LTRIM(), RTRIM(), TRIM() 

- Date: NOW(), DATE(), MONTH(), MONTHNAME(), YEAR(), DAY(), DAYNAME() 

- Querying with GROUP BY, HAVING, ORDER BY 

## **5.4 Python–SQL Interface** 

- Connecting SQL with Python using cursor 

- Performing insert, update, delete queries 

- Display data: fetchone(), fetchall(), rowcount 

- Creating database connectivity applications 

## **PHASE 6 — Advanced Computer Science** 

## **6.1 Software Engineering** 

- SDLC: introduction and steps 

- SDLC models: Waterfall, Prototype, Spiral, Agile 

- Data Warehouse & Data Mining: introduction and methodology 

## **6.2 Theory of Computation** 

- Automata: Finite Automata (FA), Deterministic FA (DFA), Non-Deterministic FA (NFA) 

- Regular Expressions and Languages 

- Context Free Grammar and Languages 

- Recursive Enumerable and Non-Recursive Enumerable Languages 

## **6.3 Compiler Design** 

- Introduction: Lexical Analysis, token recognition, minimizing DFA 

- Parsing & Parse Trees: rightmost/leftmost derivation; bottom-up and top-down parsers 

- Intermediate code generation 

- Code generation and code optimization 

## **PHASE 7 — Society, Ethics & Emerging Trends** 

## **7.1 Digital Footprints & Cyber Safety** 

- Digital society and Netizen: net, communication & social media etiquettes 

- Data protection: IPR (copyright, patent, trademark); violations; open source & licensing (Creative Commons, GPL, Apache) 

- Cyber-crime: hacking, eavesdropping, phishing, fraud emails, ransomware; prevention 

- Cyber safety: safe browsing, identity protection, confidentiality, cyber trolls and bullying 

- Safely accessing websites: malware, viruses, trojans, adware 

- E-waste management; Indian IT Act 

- Technology & Society: gender and disability issues while teaching/using computers 

## **7.2 Emerging Trends** 

- Artificial Intelligence, Machine Learning, Natural Language Processing 

- Immersive experience: AR (Augmented Reality), VR (Virtual Reality) 

- Robotics; Big Data and its characteristics 

- Internet of Things (IoT), Sensors, Smart Cities 

- Cloud Computing and Cloud Services: SaaS, IaaS, PaaS 

- Grid Computing; Blockchain technology
